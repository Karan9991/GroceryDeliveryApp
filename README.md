# Grocery Delivery App

**Delivery Boy App Features:**
- Login with mobile number
- Check all the pending order / delivered order list and details
- Accept or reject the order
- Track the order
- Cancel the order for reason like Issue with on going order, Payment issue with my order, Address wrong, other
- Contact the order person with mobile number dial-up
- Open instant map
- Deliver product with proof of customer/user signature (Signed proof of delivery)
- Check all the notifications
- Check profile and update the details anytime
- Change status with active or inactive
- Check the Order completed, Total Sales and Cancel order statics(insight)
- Receive notification once the admin assign the new orders
- Accept/deliver the order makes user gets notification
- User friendly and Unique UI
